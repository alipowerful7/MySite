درباره من

من علی خسروجردی هستم؛ برنامه‌نویس بک‌اند و عاشق تکنولوژی! تخصص من توسعه نرم‌افزار با C# و Asp.Net است. با ابزارهایی مثل Git، Github، Gitlab و Docker پروژه‌هایم را مدیریت و مستندسازی می‌کنم. عاشق تست و توسعه API با Postman و کار با دیتابیس‌های SQL Server و PostgreSQL هستم. یادگیری و حل چالش‌های جدید همیشه من را هیجان‌زده می‌کند!
